hhtml-snippets
==============

Harry's HTML snippets for Sublime Text 3.

I'm sure they will work elsewhere, but that's what I use them with.

- [x] drop (select / option)
- [x] option
- [x] check (checkbox)
- [x] submit
- [x] div
- [x] table
